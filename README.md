# Amazon-Sales-Analytics-Project
End-to-end Amazon sales data analytics project using Excel, SQL, Python, and Power BI dashboards.

## Project Overview
This is an end-to-end Data Analytics project based on Amazon sales data.  
The goal of this project is to analyze sales performance, customer behavior, product trends, and key business metrics using multiple analytical tools.

This project demonstrates skills in data cleaning, exploratory data analysis (EDA), SQL querying, data visualization, and dashboard development.

---

## Tools & Technologies Used
- Microsoft Excel
- SQL (MySQL)
- Python (Pandas, Matplotlib)
- Power BI
---

## Dataset Description
The dataset contains transactional sales records including:
- Order ID & Order Date
- Customer ID & Name
- Product Name, Category & Brand
- Quantity & Discount
- Tax, Shipping Cost & Total Amount
- Payment Method & Order Status
- City, State & Country

---

## Project Phases

### Phase 1 – Excel Analysis
- Data cleaning and formatting
- Pivot tables and summary reports
- Category-wise and country-wise analysis
- Basic charts and data dictionary creation

### Phase 2 – SQL Analysis
- Database creation and CSV import
- Aggregate queries (SUM, COUNT, AVG)
- Sales analysis by category, country, and order status
- Business insights using GROUP BY and filters

### Phase 3 – Python EDA
- Data preprocessing using Pandas
- Monthly sales trend visualization
- Sales by category comparison
- Order status distribution (Pie Chart)
- Discount vs Average Sales relationship (Scatter Plot)

### Phase 4 – Power BI Dashboards

#### Executive Overview Dashboard
- Total Sales KPI
- Total Orders KPI
- Average Order Value KPI
- Delivery Success Rate KPI
- Monthly Sales Trend
- Sales by Category
- Order Status Distribution
- Interactive Year & Category Slicers

#### Customer & Product Insights Dashboard
- Top 10 Products by Sales
- Top Customers by Sales
- Sales by Country (Map Visual)
- Payment Method Distribution
- Category Filter

---

## Key Insights
- Delivery success rate is around 75%, indicating efficient order fulfillment.
- Sales remain relatively stable with minor seasonal fluctuations.
- A small group of products contributes a large share of total revenue.
- Digital payment methods dominate customer transactions.
- Higher discounts are associated with lower average sales values.
